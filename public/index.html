shipit.

A safe, AI-assisted git CLI for commits, pushes, and pull requests.

Install

curl -fsSL https://letsshipit.vercel.app/install.sh | bash
install.sh GitHub

What it does

  • Drafts commit messages from your staged diff.
  • Reviews changes before pushing — no surprises.
  • Opens pull requests with sensible titles and bodies.
